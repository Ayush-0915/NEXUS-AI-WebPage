// what is nexus?

your always-on AI companion. 🛸

NEXUS AI is a production-grade, JARVIS-style personal AI operating system that lives on your desktop. It's not another chat window — it's an intelligent companion that communicates naturally, understands context, remembers everything important, analyzes your screen and files, executes real actions, and automates entire workflows. Always available. Always aware. Always yours.

👁️ it sees ● LIVE

Continuously captures your screen every 2 seconds, reads text via OCR, detects VS Code context, and understands what you're working on — without being asked.

📁 it knows your code ● LIVE

Scans entire project folders, detects tech stacks, maps architecture, generates READMEs, answers questions about your codebase, and can analyze itself.

📊 it monitors everything ● LIVE

Live CPU, RAM, GPU, battery and network telemetry. Hardware diagnostics, performance insights, and upgrade recommendations — all in the dashboard.

🗣️ it talks ◈ BUILDING

Natural real-time voice conversations — interrupt it mid-sentence, it handles it. Hybrid online/offline STT + TTS pipeline actively in development.

🧠 it remembers ◈ BUILDING

Long-term semantic memory using SQLite + FAISS + Sentence Transformers. Stores preferences, decisions, and past context — in development now.

⚡ it acts ◈ BUILDING

File system agent, computer control, and workflow automation — with safety checks on every action. The full planner-executor agent is next on the roadmap.
